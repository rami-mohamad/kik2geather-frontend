# Kik2Gether Frontend

Frontend for the Kik2Gether football booking application.

Users can register, log in, book football fields, and manage bookings.

## Tech Stack

- React
- React Router
- Axios
- Tailwind CSS
- Vite
- Render Deployment

## Installation

```Bash
git clone git@github.com:rami-mohamad/kik2geather-frontend.git
cd kik2gether-frontend
npm install
npm run dev
```

## Features

- Authentication

- Booking interface

- Dashboard

- Protected routes

- Booking confirmation

- Responsive design

- Mobile navigation
